# Project thumbnails (drop-in)

Drop your real University project photos here using these exact filenames:

- `autonomous-quadrotor.jpg`
- `lidar-mapping-rover.jpg`
- `5-dof-robotic-arm.jpg`

Recommended: landscape, ~800x500px, JPG or PNG. The cards crop to a 16:10-ish area (`object-cover`).

If a file is missing, the card gracefully falls back to a styled placeholder, so the site still builds and renders. To change filenames or add more projects, edit `src/app/data/site-content.ts` (the `image` field on each project).
